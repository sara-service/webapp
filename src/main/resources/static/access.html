Who can access your archived software artefact?

Everybody will be able to clone the archived git repo and continue working where you left off. You will probably not become famous, but you might get citations for your software artefact if you also create a bibliographical record. Publication cannot be undone.

The archived repo will be inaccessible. You can always make it public later on if you change your mind. Choose this option for data that you want to (or have to) keep private.

You will be given a token you can use to retrieve or publish your data. Store this token securely – you will not be able to access the archived data without it!

Creating a bibliographical record will make it possible for you and others to cite your archived software artefact. It will also make it much easier to find your software artefact, and may count as a publication in its own right.
