Who can access your archived software artefact?

Everybody will be able to clone the archived git repo and continue working where you left off. Publication cannot be undone.

The archived repo will be inaccessible. You can always make it public later on if you change your mind.

You will be given a token you can use to retrieve or publish your data. Store this token securely – you will not be able to access the archived data without it!

You will get a persistent identifier you can use to cite your archived software artefact.
